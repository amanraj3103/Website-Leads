# Dream Axis Lead Collection

Standalone lead collection web application for Dream Axis Travel Solutions. A separate frontend and Flask backend system that collects enquiries for education and job placement services.

## What it does

- Dynamic web form with service-specific fields (Education India, Education Abroad, Job Europe)
- Real-time form validation with helpful error messages
- Flask backend API that processes and stores submissions in Google Sheets
- Responsive design matching the main Dream Axis website
- Smooth animations and transitions for a polished user experience
- Deployed as an independent application on Vercel

## Tech stack

HTML, CSS, JavaScript, Python, Flask, Google Sheets API, Vercel
